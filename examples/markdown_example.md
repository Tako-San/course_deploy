## Codeblock

Нужно уметь писать куски кода на языке С внутри строки, например `int * px = &x;` и все будет хорошо.

Не надо надеяться, что *px и &x будут правильно сформатированы.

Как и **argc, например.

Программа на python:
```python
print('Hello, world!')
```

Программа на С:
```cpp
#include <stdio.h>

int main(int argc, char * argv[]) {
	printf("Hello, world!\n");
	return 0;
}
```
